# Glossário

Este glossário reúne os termos mais recorrentes no contexto do projeto Toque de Mulher.

- **Cliente:** Láercia Holanda, responsável por validar requisitos e aprovar entregas.
- **Usuário:** consumidor final que navega, compra e interage com a plataforma.
- **MVP (Produto Mínimo Viável):** primeira versão funcional focada no valor central do negócio.
- **Painel administrativo:** interface voltada à operação da loja, como produtos, estoque, pedidos e fornecedores.
- **Loja virtual:** interface pública usada por clientes finais.
- **Automação operacional:** redução de tarefas manuais na rotina comercial.
- **Gateway de pagamento:** serviço externo responsável por processar pagamentos online.
- **SEO:** otimização para motores de busca, usada para descoberta orgânica da loja.
- **KPI:** indicador-chave de desempenho usado para medir resultado de negócio.
- **AOV (Average Order Value):** valor médio por pedido.
- **CAC (Customer Acquisition Cost):** custo de aquisição de cliente.
- **LTV (Lifetime Value):** valor total gerado por um cliente ao longo do relacionamento.
- **PWA (Progressive Web App):** aplicação web com comportamento próximo ao de app nativo.
- **Headless:** arquitetura em que frontend e backend ficam desacoplados e se comunicam por API.
- **UGC (User Generated Content):** conteúdo criado pelos próprios usuários, como avaliações com foto e vídeo.
- **LGPD:** Lei Geral de Proteção de Dados, aplicável ao tratamento de dados pessoais.
- **Routine Builder:** funcionalidade planejada para montar rotinas personalizadas de beleza.
- **Wishlist social:** lista de desejos compartilhável, usada para retenção e aquisição orgânica.
