# Objetivos do Projeto

Os planos do projeto organizam os objetivos em quatro frentes estratégicas.

## Objetivos estratégicos

| Objetivo | Resultado esperado |
| :--- | :--- |
| Excelência operacional | Reduzir em até 80% o tempo gasto em processos manuais. |
| Expansão de mercado | Levar a operação da loja para alcance regional e nacional. |
| Diversificação de receita | Abrir um canal digital complementar às vendas físicas. |
| Fortalecimento da marca | Consolidar a Toque de Mulher como referência confiável em cosméticos de curadoria. |

## Objetivos funcionais do MVP

O MVP deve entregar, no mínimo:

- loja virtual pública para navegação, descoberta e compra de produtos;
- painel administrativo para gestão de produtos, estoque, pedidos e fornecedores;
- cadastro e autenticação de usuários;
- catálogo com categorias, filtros, imagens e descrições;
- carrinho e checkout com integração a gateway de pagamento;
- relatórios básicos para acompanhamento de vendas;
- experiência responsiva para desktop e mobile;
- base mínima de SEO para descoberta orgânica.

## Indicadores-chave de desempenho

As metas levantadas nos documentos-fonte são:

| Indicador | Meta inicial |
| :--- | :--- |
| Taxa de conversão | Superior a 2,0% |
| Abandono de carrinho | Inferior a 75% |
| Crescimento mensal da receita online | Igual ou superior a 10% |
| Redução do esforço manual | Até 80% |
| Novos clientes registrados | Meta a acompanhar no primeiro ciclo de operação |

## Critério de sucesso

O projeto será considerado bem-sucedido quando a plataforma permitir operação comercial real, com redução do trabalho manual, aumento de alcance da loja e base técnica pronta para evoluir com recursos de fidelização, gamificação e personalização.
