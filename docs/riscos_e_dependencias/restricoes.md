# Restrições do Projeto

As principais restrições identificadas para o projeto são:

- disponibilidade variável da equipe por se tratar de uma iniciativa com forte participação voluntária;
- necessidade de manter o escopo do MVP enxuto para evitar atraso de entrega;
- dependência de aprovação da cliente para decisões visuais, funcionais e de priorização;
- orçamento inicial limitado, com foco em ferramentas gratuitas e infraestrutura de baixo custo;
- dependência de serviços de terceiros para pagamento e hospedagem;
- necessidade de manter compatibilidade entre frontend, backend e documentação, o que hoje ainda exige alinhamento.
